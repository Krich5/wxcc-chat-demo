IMI / Webex Chat Widget — Minimal Demo

This page loads the widget script directly (no XHR) to avoid CORS/origin problems.

Instructions

  1. Serve this file from a local web server (not file://). For example:
    • python -m http.server 8080 then open http://localhost:8080/chat-demo.html
  2. Open your browser’s DevTools Console if you don’t see the launcher; copy any errors to me.

If your tenant limits allowed origins, make sure http://localhost:8080 (or whatever you use) is allow-listed.
